# Chrome QR Code Scanner

一个纯离线、轻量级的 Chrome 浏览器扩展插件，能够让你在浏览器网页上直接框选任意区域并快速识别其中的二维码，省去保存图片再解析的繁琐步骤！

## ✨ 核心特性

- **一键框选截屏**：无复杂界面，点击插件图标立即在当前网页进行拖拽框选。
- **本地纯前端解析**：核心使用 `jsQR` 通过 Offscreen Document (Canvas) 解析，100% 本地运算，无需任何网络请求或后端服务支持，保护用户的隐私数据安全。
- **自动剪贴板复制**：一旦成功识别到二维码，工具将会把解码内容瞬间复制到你的系统剪贴板。
- **贴心验证与跳转**：如果是网址链接 (URL)，结果弹窗会提供“在新标签页打开”的一键跳转。

## 📦 安装指南

由于当前应用未发布至 Chrome 网上应用店，你需要通过“开发者模式”来手动加载它。

1. **下载源码**
   将本仓库代码 Clone 或下载 ZIP 到你的本地磁盘。

2. **进入 Chrome 扩展管理**
   在浏览器地址栏输入 `chrome://extensions/` 按回车。

3. **开启开发者模式**
   在页面的 **右上角**，找到“开发者模式”开关并将其打开。

4. **加载插件**
   点击左上角的“**加载已解压的扩展程序**”，然后在弹出的窗口中选择本项目源码存放的文件夹（包含 `manifest.json` 的目录）。

5. **固定使用** (可选)
   加载成功后，点击浏览器右上角的拼图块图标 🧩，找到 `QR Code Scanner`，点击图钉 📌 将它固定到浏览器工具栏。

## 🚀 使用方式

1. 打开任何展示了二维码的网页。
2. 点击工作栏右上角的本扩展程序图标 📱。
3. 网页会变暗，此时你的鼠标会变成十字准心模式。
4. **按住鼠标左键** 框选你的二维码。
5. 选区完成松开鼠标，界面会立刻展示一张放大裁减预览图及下方解码出来的实际文本（同时后台已帮你偷偷复制到剪贴板中！可以直接 `Ctrl+V`）。
6. 解析失败的话支持重新尝试，直接按按键盘 `ESC` 可随时退出扫描模式。

## 🔧 技术栈与清单
- **Chrome Extension API** (`Manifest V3`)
- **jsQR** (二维码像素解析的核心推断库)
- Vanilla JavaScript & CSS3 (原生的 UI 构建交互)
